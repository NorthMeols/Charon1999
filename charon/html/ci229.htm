To use this system

  • Press the button "Select login Window"
    Log in with your name and password
    This is the same as your login name and password on the system ermintrude.
    You may need to register on this system.

  • Press the button "Select Submit Window"
    Paste your program into the window
    Change the exercise number (default 1) to the exercise you wish to submit.
    Press the "Submit exercise" button.

  • Press the button "Select progress Window"
    Press the "Examine progress" button.

All transactions to this system are logged.
